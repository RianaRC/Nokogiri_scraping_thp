## Pour commencer

Vous devez avoir  `nokogiri` sur votre ordinateur:

```shell
gem install nokogiri
```






## Pour lancer le fichier 

Dans le shell allez dans le dossier ou votre fichie se situe puis :

```shell
ruby fichier.rb #pour tester
```
